URL d'accueil : http://localhost:3000/auth.html

Pour afficher les évaluations, il faut cliquer D'ABORD sur une session, PUIS sur un étudiant, sans quoi le serveur crashera. 

La première fois qu'on évalue un étudiant sur une session particulière, il faut effectuer la manip [clic sur une session -> clic sur un étudiant] DEUX FOIS pour afficher l'évaluation.
